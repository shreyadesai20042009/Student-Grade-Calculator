# 🎓 Grade Calculator (Java)

A simple Java console-based application that calculates total marks, average percentage, and assigns grades based on user input.

This project demonstrates fundamental Java concepts such as arrays, loops, conditional statements, and user input handling.

---

## 📌 Problem Statement

Create a program that:

- Accepts marks for multiple subjects
- Calculates total marks
- Computes average percentage
- Assigns a grade based on percentage
- Displays final result

---

## 🛠 Tech Stack

- Java
- Scanner Class (User Input)
- Arrays
- Conditional Statements
- Loops

---

## 🚀 Features

✅ Takes dynamic number of subjects  
✅ Stores marks using arrays  
✅ Calculates total marks  
✅ Computes average percentage  
✅ Assigns grade automatically  
✅ Clean and simple console output  

---

## 🧠 Grade Criteria

| Percentage Range | Grade |
|------------------|-------|
| 90 and above     | A     |
| 80 – 89          | B     |
| 70 – 79          | C     |
| 60 – 69          | D     |
| Below 60         | F     |
